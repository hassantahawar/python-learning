# 🟦 PHASE 0 — Python for AI Engineers (Month 1)

### 🎯 Goal

Become **comfortable writing production Python**, not academic scripts.

---

## What to Learn (VERY SPECIFIC)

### Python Core (Must)

* Data types
* Functions
* Loops
* Error handling
* Modules & packages
* Virtual environments

### Python OOP (LIMITED)

You **DO need OOP**, but:

* Classes
* `__init__`
* Composition > inheritance
* Dataclasses

❌ Skip:

* Design patterns
* Multiple inheritance
* Metaclasses

---

### Libraries (AI-focused)

* `requests`
* `pydantic`
* `dataclasses`
* `typing`
* `logging`

---

## Daily Breakdown (Example Week)

**Day 1:** Python basics
**Day 2:** Functions + errors
**Day 3:** OOP essentials
**Day 4:** File handling + JSON
**Day 5:** Typing + Pydantic
**Day 6:** Mini script project

---

## Mini Project

> CLI app that:

* Reads a prompt from file
* Calls an LLM API (mock)
* Saves response to JSON

---

## ✅ Phase-Completion Checklist

Move on ONLY if:

* [ ] You can write Python without Googling syntax
* [ ] You understand classes & composition
* [ ] You can read others’ Python code
* [ ] You can structure a small project
